## How to take this course

Each chapter ends with a **Checklist** (things you should be able to do or identify). The end of each section has a **Review** page with a short quiz that feeds the Lernkit Tracker — those answers are what would be reported to an LMS via SCORM / xAPI in a real delivery.

Everything in this course is **self-paced**. The code blocks are copy-pasteable. The `.robot` snippets run on your own machine against your own Robot Framework install — there's no remote runner yet (that lands with the Phase-3 rf-mcp integration per [ADR 0009](https://github.com/manykarim/lernkit/blob/main/docs/adr/0009-reuse-rf-mcp-as-robot-framework-runner-base.md)).

## What this course is not

- It's **not** a replacement for the [Robot Framework User Guide](https://robotframework.org/robotframework/latest/RobotFrameworkUserGuide.html). The UG is the authoritative reference; this course is a guided first path through it.
- It's **not** a browser-automation tutorial. That's a follow-on course covering the Browser library (Playwright-based) specifically.
- It's **not** a certification course. There's no proctored exam at the end, and the quizzes are formative, not gated.
