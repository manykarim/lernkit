These exercises give you a chance to practise what you've learnt on a realistic logistics scenario. Each one ships with a starter `.robot` skeleton — edit it in the browser and hit **Run suite** to see if it passes.

<Aside type="note">
  Each starter code below has a placeholder (`# TODO: ...`) where you fill in your solution. Lernkit doesn\u2019t yet ship a native `<Exercise>` component with hidden auto-grading tests — that lands with the Phase-3 rf-mcp integration (see [ADR 0009](https://github.com/manykarim/lernkit/blob/main/docs/adr/0009-reuse-rf-mcp-as-robot-framework-runner-base.md)). For now, the built-in assertions tell you when you got it right.
</Aside>

## Exercise 2.1 — Warehouse Welcome (★☆☆, ~5 min)

You've arrived for your first shift at the Munich Distribution Center. Log your arrival and assert the message is correct.

**Keywords to use:** `Log`, `Log To Console`, `VAR`, `Should Not Be Empty`

## Exercise 2.2 — Shipment Inspector (★★☆, ~10 min)

A new shipment has arrived. Build a dictionary with its details, flag it if it's hazardous, and iterate over all its fields.

**Keywords to use:** `Create Dictionary`, `Should Contain`, `Should Not Contain`, `Set To Dictionary`, `IF / ELSE / END`, `FOR / IN / END`

<RunnableRobot
  client:visible
  fileName="shipment_inspector.robot"
  caption="The skeleton has the imports and test structure. Fill in the dict, the asserts, and the IF + FOR blocks."
  initialCode={`*** Settings ***
Documentation    Exercise 2.2 — Shipment Inspector
Library          Collections

*** Test Cases ***
Inspect Shipment
    # TODO: Build &{SHIPMENT} with origin=Munich, destination=Amsterdam,
    #       weight_kg=\${8200}, hazardous=\${FALSE}.
    &{SHIPMENT}=    Create Dictionary    origin=Munich    destination=Amsterdam
    ...    weight_kg=\${8200}    hazardous=\${FALSE}

    # TODO: Assert it CONTAINS 'origin' but does NOT contain 'tracking_id'
    Dictionary Should Contain Key        \${SHIPMENT}    origin
    Dictionary Should Not Contain Key    \${SHIPMENT}    tracking_id

    # TODO: IF hazardous, log a warning. ELSE, log "Standard cargo".
    IF    \${SHIPMENT}[hazardous]
        Log    ⚠ Hazardous cargo
    ELSE
        Log    Standard cargo
    END

    # TODO: Add tracking_id=SHP-2026-0042 using Set To Dictionary.
    Set To Dictionary    \${SHIPMENT}    tracking_id=SHP-2026-0042
    Dictionary Should Contain Key    \${SHIPMENT}    tracking_id

    # TODO: Loop over all key-value pairs and log each.
    FOR    \${k}    \${v}    IN    &{SHIPMENT}
        Log    \${k} = \${v}
    END
`}
/>
