The Lernkit repo documents three layers of decisions. Read them in this order when you need depth.

## ADRs

Architectural decisions live in `docs/adr/` and follow [MADR 3.0](https://adr.github.io/madr/). Each ADR captures **one** decision, the alternatives considered, and the consequences accepted.

See the full index at [`docs/adr/README.md`](https://github.com/manykarim/lernkit/blob/main/docs/adr/README.md).
