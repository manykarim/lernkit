## Phase 2 preview

In Phase 2 this page will render a `<RunnablePython>` component backed by Pyodide 0.29.x in a Web Worker. The code below will be executable in-browser:

```python
import sys

print(f"Python {sys.version} running in Pyodide")
print("State persists across cells in the same cellGroup.")
```

For now it is a static snippet.

## How it will work (preview)

- Pyodide loads lazily on first `▶ Run` click.
- State persists within a `cellGroup` via a shared `py.globals` dict.
- Outputs stream back via `postMessage`; stdout and stderr are distinguished.
- Every execution emits an xAPI `executed-code` statement via the `Tracker` interface ([ADR 0004](https://github.com/manykarim/lernkit/blob/main/docs/adr/0004-unified-tracker-interface-with-pluggable-adapters.md)).
- Source code is stored as a SHA-256 hash on intermediate runs; full source is kept only on terminal `passed` / `failed` events.
