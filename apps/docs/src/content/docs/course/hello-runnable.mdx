## Run some Python

The cell below is a React island. The first click on **Run** boots Pyodide 0.29.3 in a Web Worker — expect a ~2–5 s warm-cache delay. Subsequent runs on the same page are fast.

<RunnablePython
  client:visible
  caption="Try changing the numbers, then hit Run again."
  initialCode={`import sys

def squares(n):
    return [i * i for i in range(n)]

print(f"Python {sys.version_info.major}.{sys.version_info.minor} via Pyodide")
print(squares(8))
`}
/>

## What just happened

- The component lazy-loads a Web Worker on first click (protects LCP on prose pages).
- The worker dynamically imports **self-hosted** `pyodide.mjs` from `/pyodide/` (per [ADR 0006](https://github.com/manykarim/lernkit/blob/main/docs/adr/0006-pyodide-in-web-worker-for-in-browser-python.md) — never jsDelivr).
- stdout and stderr are captured via Pyodide's `setStdout` / `setStderr` batched callbacks.
- Uncaught exceptions surface as a formatted error panel, not a browser crash.

## Try an error

```python
# This will raise — you'll see the type + message in the error panel.
def divide(a, b):
    return a / b

print(divide(10, 0))
```

Paste that into the editor above and run. The error panel shows the Python type and message.

## Phase 2 limits still in force

- No `input()` yet — requires cross-origin isolation (COOP/COEP). Planned for a dedicated `/run/` route per [ADR 0019](https://github.com/manykarim/lernkit/blob/main/docs/adr/0019-scope-cross-origin-isolation-headers-to-runner-page.md).
- No on-demand package loading (numpy, pandas, matplotlib). Pyodide's default `loadPackagesFromImports` would fetch wheels from the Pyodide CDN, which violates the self-host policy. A mirror arrives in Phase 3.
- No persistent `cellGroups` across multiple `<RunnablePython>` cells on one page — each renders an isolated worker. Planned for Phase 2+.
- No interrupt button. Runaway loops require a page reload for now.
- No xAPI emit yet for the execution itself. Phase 3 wires `executed-code` / `passed` / `failed` statements for `<CodeChallenge>` (the autograded sibling of this widget).

See [`docs/plan/02-phase-plan.md`](https://github.com/manykarim/lernkit/blob/main/docs/plan/02-phase-plan.md) for the full Phase 2 + Phase 3 scope.
